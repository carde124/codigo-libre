
## 🎨 Colores Utilizados

La paleta de colores ha sido diseñada para **impactar al ser humano** con máxima profesionalidad:

- **Cyan/Turquesa (#00ffff)**: Representa tecnología, innovación y modernidad
- **Púrpura/Magenta (#ff00ff)**: Creatividad, lujo y sofisticación
- **Azul Oscuro (#1a0033, #220055)**: Confianza, profesionalismo y seguridad
- **Blanco (#ffffff)**: Claridad y limpieza
- **Gris Claro**: Complemento profesional

## 🎯 Personalización

### Cambiar Información de Contacto

Edita en `index.html`:

```html
<!-- Email -->
<p>info@codigolibre.ai</p>

<!-- Teléfono -->
<p>+34 (XXX) XXX-XXXX</p>

<!-- Redes Sociales -->
<a href="tu-url-linkedin"><i class="fab fa-linkedin"></i></a>
```

### Cambiar Imágenes de Galería

Reemplaza las URLs en las imágenes de la sección galería:

```html
<img src="TU_NUEVA_IMAGEN_URL" alt="Descripción">
```

Puedes usar:
- URLs externas de servicios como Unsplash, Pexels
- Tus propias imágenes generadas por IA
- Rutas locales: `src="images/proyecto1.jpg"`

### Ajustar Colores

En `styles.css`, busca estas variables:

```css
background: linear-gradient(135deg, #00ffff, #ff00ff);  /* Gradiente Cyan-Púrpura */
color: #ff00ff;  /* Púrpura */
color: #00ffff;  /* Cyan */
background: linear-gradient(135deg, #1a0033 0%, #220055 100%);  /* Fondo oscuro */
```
