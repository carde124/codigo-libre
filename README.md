# Código Libre - Proyecto Local

Este repositorio contiene una página web estática y un servidor Node.js simple para servirla localmente.

Estructura de archivos principal:

- `index (2).html` — Página principal.
- `styles (1).css` — Estilos CSS.
- `script (1).js` — JavaScript del frontend.
- `server (1).js` — Servidor Node.js que sirve los archivos.

Cómo ejecutar el proyecto localmente:

1. Abre una terminal en la carpeta del proyecto.

2. Ejecuta el servidor con Node.js:

```bash
cd "c:\Users\Usuario\web ia mia"
node "server (1).js"
```

3. Abre tu navegador en `http://localhost:3000/`.

Notas:

- El servidor incluye decodificación de URLs para manejar nombres de archivos con espacios y paréntesis (por ejemplo `styles (1).css`).
